## HR Analytics Dashboard (Power BI)

### Objective
Analyze employee attrition and workforce trends using Power BI.

### Features
- Attrition count and attrition rate KPIs
- Department, education, and age-based analysis
- Interactive slicers and drill-down
- DAX measures and conditional formatting

### Tools Used
- Power BI
- DAX
- Excel

### How to Use
1. Download the Dashboard.pbix file
2. Open it in Power BI Desktop
